# Changelog

All notable changes to DorkNexus will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-08

### Added
- **Dork Builder**: Visual query constructor with 7 input fields (site, filetype, intitle, inurl, intext, exact match, exclude)
- **AI Dork Generator**: Natural language to dork conversion powered by Gemini 3 Pro
- **Deep Analyzer**: AI-powered strategy analysis with thinking mode and risk assessment
- **Template Gallery**: 60+ pre-built dork templates across 4 categories
- **Multi-Engine Pivot**: Query translation for Shodan, Censys, Hunter.io, and ZoomEye
- **DorkResearch Hub**: AI-grounded topic research with live search data
- **NexusTerminal**: Simulated terminal interface with live search capabilities
- **NexusVault**: Local persistent storage for saving and organizing queries
- **Voice Command Center**: Hands-free voice input support with microphone integration
- **Video Generator**: AI-generated educational video tutorials via Veo 3.1
- Real-time query preview with sticky input bar
- One-click copy to clipboard functionality
- Direct Google search integration
- Dark theme cybersecurity-focused UI
- Responsive mobile-friendly design
- TypeScript strict mode support

### Technical
- React 19.2.3 with modern hooks
- Vite 6.2.0 build system
- Google Gemini API integration (@google/genai)
- Tailwind CSS styling via CDN
- Lucide React icon library
